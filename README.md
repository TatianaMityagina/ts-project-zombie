# Zombie

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

## Task
```
1. Глубокое погружение Vue-Router
   Цель: Научиться решать задачи с применением всех возможностей vue-router

Ожидаемый результат:
1) Создание динамических роутов в зависимости от роли пользователя

2) Реализовать на каждом роуте отдельную проверку доступности роута для пользователя 
(если пользователю не доступен роут, то при переходе на него его перекидывает на страницу 404)

2. Глубокое погружение Vuex
  Цель: Научиться решать задачи с применением всех возможностей vuex

  Ожидаемый результат:
 1) Разбивка vuex на модули
 2) Созданные модули перед выполнением action должны использовать функцию проверки 
 возможности выполнения действия использую функцию проверки из рут стора
 3) Во всех компонентах доступ к стору должен выполнятся с использованием маперов

 4) Реализовать динамическую регистрацию модулей (модули становятся доступными 
  vuex просто добавив файл в папку modules)

3. Создание кастомных Vue компонентов
  Цель: Научится применять возможности Vue для создания компонентов и оберток над внешними библиотеками

  Ожидаемый результат:
 1) Созданный компонент Button
 2) Созданный компонент Input  

 3) Созданный компонент Checkbox

Примечание: Все компоненты должны иметь поведение стандартных тегов html, 
в компонентах запрещено использовать секцию props

4. Погружение в TS
  Цель: Освоить базовые возможности ts

  Ожидаемый результат:
 1) Ознакомится с возможностями ts

 2) Каждая сущность используемая в проекте должна быть типизирована
 3) Каждая функция используемая в проекте должна быть типизирована 
 и явно указывать что он возвращает или выполняет
```
